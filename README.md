# FullstackII EP2 
Profesor aca le dejo los comandos que usamos para las pruebas unitarias y para ver la pagina

Instalar y correr:

npm install

npm run dev (abre la URL que muestra Vite)

npm run test (pruebas unitarias + coverage)

Credenciales:
para ingresar como admin a la pagina
admin / admin

También puede registrarse.

Notas:

/admin solo para rol “Administrador”.

Coverage HTML: start coverage\html\index.html.
